# 77 TEAM — Reconstrução organizada

Estrutura modular:

- `firebase-config.js`: configuração Firebase
- `auth.js`: mensagens e autenticação
- `roles.js`: clãs, cargos e badges
- `store.js`: estado central
- `attendance.js`: presença e estatísticas
- `render.js`: renderização
- `main.js`: inicialização e integrações
- `ui.js`: navegação
